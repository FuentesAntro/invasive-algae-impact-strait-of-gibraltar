# Invasive Algae Impact — Strait of Gibraltar
**Mixed-methods analysis of socio-environmental conflict in three fishing communities**
Barbate, Conil and Tarifa · March 2026

## Overview
This project analyses the socio-environmental impact of an invasive algae species 
on artisanal fishing communities in the Strait of Gibraltar, based on 198 qualitative 
citations extracted from ethnographic interviews.

## Methods & Tools
- **ATLAS.ti** — ethnographic coding, co-occurrence analysis, concept networks
- **R / RStudio** — data visualisation (ggplot2, sankeyD3, factoextra)

## Key Outputs
- Co-occurrence matrix across 6 conflict categories
- Sankey diagram: impact flow by fishing port
- Conflict interconnection network
- Correspondence map: ports vs. conflict themes

## Main Finding
The invasive algae is not an isolated environmental problem — it triggers a domino 
effect that collapses fishing grounds and directly threatens the economic survival 
of coastal families, with Barbate being the most critically affected port.
